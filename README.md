# ☕ Java & Spring Boot Learning Journey

Welcome to my Java and Spring Boot learning repository! This workspace is dedicated to tracking my progress, organizing university coursework, and practicing core Java and advanced Spring Boot concepts.

---

## 📂 Repository Structure

The repository is organized into two main sections:

### 🏛️ University Learning (`/University-Learning`)
*   **Purpose:** This folder contains all the materials, notes, and topics covered in my university classes.
*   **Structure:** Organized by class numbers (e.g., `Class-01`, `Class-02`, etc.) to match the academic curriculum.

### 🚀 Self-Paced Practice (`/phase-XX`)
*   **Purpose:** These folders are dedicated to step-by-step self-paced learning and practical implementations.
*   **Pathways:**
    *   `phase-01-Foundation` - Basic Java syntax, compilation, and structure.
    *   `phase-02-variables-data-types` - Variables, operations, and primitive data types.
    *   `phase-03-input-conditions` - User input (Scanner) and conditional logic.
    *   `phase-04-loops` - Iterations (for, while, do-while loops).
    *   `phase-05-arrays-strings` - Array manipulation and String handling.
    *   `phase-06-methods` - Writing functions, reusable code blocks, and parameters.
    *   `phase-07-oop` - Object-Oriented Programming (Classes, Objects, Inheritance, Polymorphism, Abstraction, Encapsulation).
    *   `phase-08-exception-file-handling` - Error handling and reading/writing files.
    *   `phase-09-collections` - Lists, Sets, Maps, and data structures.
    *   `phase-10-projects` - Real-world project implementations (e.g., Student Management, Bank Management).

---

## 🛠️ How to Run locally

To compile and run any Java file (e.g., `hello.java`):

1. **Compile:**
   ```bash
   javac phase-01-Foundation/practice/hello.java
   ```
2. **Run:**
   ```bash
   java phase-01-Foundation/practice/hello
   ```
