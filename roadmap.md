# 🗺️ Java Learning Roadmap

This roadmap is divided into phases. Each phase has three parts:
- 📖 **What to learn**: The theoretical concepts you need to grasp.
- 💻 **What to practice**: The code you need to write to solidify your understanding.
- ✅ **What to complete**: The tasks you need to finish before moving to the next phase.

---

## 📌 Phase 01: Java Foundation

### 📖 Learn
- What Java is
- JDK, JRE, JVM
- Compile vs run
- Java program structure
- `main()` method
- `System.out.println()`

### 📝 Tasks
- [ ] Install JDK
- [ ] Install VS Code or IntelliJ
- [ ] Run first Java program
- [ ] Understand class and `main` method
- [ ] Write 5 simple print programs

### 💻 Practice
- Print your name
- Print your university name
- Print your department
- Print multiple lines
- Print formatted output

### ✅ Done when...
- You can create and run a Java file without help
- You understand the role of JDK and JVM

---

## 📌 Phase 02: Variables, Data Types, Operators

### 📖 Learn
- Variables
- Primitive data types
- Non-primitive data types
- Type casting
- Arithmetic operators
- Relational operators
- Logical operators
- Assignment operators
- Unary operators
- Ternary operator

### 📝 Tasks
- [ ] Declare variables with proper naming
- [ ] Use each primitive data type
- [ ] Convert between compatible types
- [ ] Solve operator expressions

### 💻 Practice
- Store age, name, cgpa, grade, and status
- Build a calculator using variables
- Compare two numbers
- Check even or odd
- Use ternary operator in a small program

### ✅ Done when...
- You can choose the correct data type without guessing
- You can write small operator-based programs independently

---

## 📌 Phase 03: Input and Conditionals

### 📖 Learn
- Scanner
- User input
- `if`, `if-else`, `else-if`, nested `if`

### 📝 Tasks
- [ ] Take input from user

- [ ] Handle simple decisions
- [ ] Compare multiple conditions

### 💻 Practice
- Odd or even checker
- Largest of two numbers
- Largest of three numbers
- Grade calculator
- Simple login validation
- Day name from number using `switch`

### 📝 Tasks

- [ ] Solve repetition problems

### 📝 Tasks

- Return type and `void`

### 📝 Tasks

### 💻 Practice

## 📌 Phase 10: Projects

### 🚀 Projects to Build
1. **Project 1: Calculator** (Skills: variables, input, conditionals, methods)
2. **Project 2: Student Management System** (Skills: OOP, arrays, methods)
3. **Project 3: Library Management System** (Skills: OOP, collections, search logic)
4. **Project 4: Bank Management System** (Skills: OOP, exception handling, file handling)
5. **Project 5: Restaurant Management System** (Skills: OOP, collections, menu logic)

### 📝 Project Tasks
- [ ] Define features clearly
- [ ] Design classes first
- [ ] Break work into small functions
- [ ] Test each feature separately
- [ ] Fix bugs before adding more features

---

## 📅 Daily Study Rule
- Study one topic at a time
- Write code every day
- Do not just watch tutorials
- Practice at least 5 small problems per day
- Revise old topics weekly

## 🔄 Suggested Weekly Flow
- **Day 1**: Learn concept
- **Day 2**: Write basic code
- **Day 3**: Solve easy problems
- **Day 4**: Solve medium problems
- **Day 5**: Revise and fix mistakes
- **Day 6**: Build mini task
- **Day 7**: Review the whole phase

## 🎯 Final Goal
After finishing all phases, you should be able to:
- Write Java programs confidently
- Understand OOP clearly
- Solve beginner to intermediate problems
- Build small projects without depending on copied code
